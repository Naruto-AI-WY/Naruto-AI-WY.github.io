Mac下使用Hugo+Github托管搭建个人博客

2020-03-10
pages

文章目录

疫情期间，时间比较多，一直想着写点自己的点滴，也没什么特别的主题，当做是日常记录吧，具体后面写点啥，后面慢慢开脑洞吧。第一篇就留给博客网站搭建纪实了。

安装Hugo：

Hugo号称世界上最快的建站框架，由go语言开发。 Mac下直接使用 Homebrew 安装：

1
2
3
4
5

brew install hugo
# 检查安装成功  
hugo version 

Hugo Static Site Generator v0.66.0/extended darwin/amd64 BuildDate: unknown

注意：如果发现使用brew命令，却总是提示：-bash: brew: command not found，则是你的mac上homebrew没有安装

1

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

建立个人博客目录：

1
2
3
4
5
6

hugo new site your_blog_name  # 此处your_blog_name会生成一个同名的文件夹，里面是此次的project site文件
cd your_blog_name  
git init  #初始化本地数字仓库

# 目录结构，tree命令 
tree blog
  • config.toml 是配置文件，在里面可以定义博客地址、构建配置、标题、导航栏等等。
  • themes 为放置主题的文件夹，可以在themes.gohugo.io 查看喜欢的主题。另外如果更改了已应用的主题的src文件夹下的内容，需要重新build主题
  • content 是存放博客内容的文件，支持任何级别的文件夹和文件的嵌套，最后的文件系统里 markdown 文件路径和站点 url 的对应关系一般如下： ``` └── content └── about | └── _index.md // <- https://example.com/about/ ├── post | ├── firstpost.md // <- https://example.com/post/firstpost/ | ├── happy | | └── ness.md // <- https://example.com/post/happy/ness/ | └── secondpost.md // <- https://example.com/post/secondpost/ └── quote ├── first.md // <- https://example.com/quote/first/ └── second.md // <- https://example.com/quote/second/

有两个文件名名字是有特殊含义的： _index.md 和 index.md 可以用于组织页面，更多请参考 Hugo 官方文档。  
- <font color=orange>static</font>包括了 css, js, fonts, media 等，决定网站的外观  
- <font color=orange>archetypes</font>包含了default.md文件，是新建博客文件时使用的模板，可以custom模板。
## 下载主题
这里我选择了even主题，可以根据喜好选择。[even主题](https://themes.gohugo.io/hugo-theme-even/)  
把github上的主题直接clone到你的博客文件夹的themes目录下，这样对主题的改动也可以同时提交到git控制中。

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

1

也可以添加到 git 的 submodule 中，优点是后面讲到用 travis 自动部署时比较方便。如果需要对主题做更改，最好fork主题再做改动。

git submodule add https://github.com/olOwOlo/hugo-theme-even.git themes/even

1
2
3

注：<font color=red>如果git clone不成功或者速度过慢的话，可以考虑搜索一下更改hosts文件的方法或者使用代理服务器，可自行搜索</font>  
### 重新build主题
更改了主题的话，需要切换到 themes/even 目录下重新 build。

cd themes/even && npm i && npm start

1

## 开始写文章

hugo new post/my-first-post.md ``` 文章顶部是一些 meta 信息，可以更改。也可以使用一些固定模板自动生成 meta 信息，更改你的博客文件目录下的archetypes文件夹里面的default.md的文件
